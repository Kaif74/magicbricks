ğŸ’¡ How It Works

  1. User selects a city from the home page
  2. Navigation to /city/[cityName]
  3. Server-Sent Events stream starts from /api/stream-scrape
  4. Puppeteer scrapes MagicBricks in the background
  5. Properties appear incrementally on the left panel
  6. Each property is geocoded using PositionStack API
  7. Map markers update as coordinates are obtained
  8. Interactive features: Click cards to highlight markers, click markers to scroll to cards
